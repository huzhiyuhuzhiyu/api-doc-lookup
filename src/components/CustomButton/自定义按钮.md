# CustomButton 自定义按钮组件

CustomButton 是一个通用的按钮组件，用于根据权限动态渲染不同类型的按钮，包括普通按钮和导出按钮。

## 功能特性

- 根据权限控制按钮显示/隐藏
- 支持普通按钮和导出按钮两种类型
- 支持插槽自定义按钮内容
- 支持按钮点击事件统一处理

## 使用方法

### 基本用法

```vue
<template>
  <CustomButton
    :btn-list="buttonList"
    :export-disabled="exportDisabled"
    :export-list-query="exportQuery"
    @click="handleButtonClick"
  />
</template>

<script>
import CustomButton from '@/components/CustomButton'

export default {
  components: {
    CustomButton
  },
  data() {
    return {
      buttonList: [
        {
          buttonType: 'primary',
          type: 'add',
          text: '新增',
          icon: 'el-icon-plus',
          permission: 'btn_add'
        },
        {
          buttonType: 'success',
          type: 'export',
          text: '导出',
          icon: 'el-icon-download',
          permission: 'btn_export',
          exportType: 1,
          exportName: '数据导出',
          tableRef: 'dataTable'
        }
      ],
      exportDisabled: false,
      exportQuery: {
        // 导出查询参数
      }
    }
  },
  methods: {
    handleButtonClick(type) {
      switch (type) {
        case 'add':
          // 处理新增逻辑
          break
        // 其他按钮类型处理
      }
    }
  }
}
</script>
```

## 属性 (Props)

| 属性名 | 类型 | 必需 | 默认值 | 说明 |
|-------|------|-----|--------|------|
| btnList | Array | 是 | - | 按钮配置列表 |
| exportDisabled | Boolean | 否 | false | 导出按钮是否禁用 |
| exportListQuery | Object | 否 | {} | 导出按钮的查询参数 |

### btnList 项配置

每个按钮配置项包含以下属性：

| 属性名 | 类型 | 必需 | 说明 |
|-------|------|-----|------|
| buttonType | String | 是 | 按钮类型 (primary/success/info/warning/danger/text) |
| type | String | 是 | 按钮唯一标识 |
| text | String | 是 | 按钮显示文本 |
| icon | String | 否 | 按钮图标 |
| size | String | 否 | 按钮大小，默认为 'mini' |
| permission | String | 否 | 权限标识，用于控制按钮显示 |
| disabled | Boolean | 否 | 是否禁用(仅导出按钮有效) |
| exportType | Number | 否 | 导出类型(仅导出按钮有效) |
| exportName | String | 否 | 导出文件名(仅导出按钮有效) |
| tableRef | String | 否 | 表格引用名称(仅导出按钮有效) |

## 事件

| 事件名 | 参数 | 说明 |
|-------|------|------|
| click | (type: String) | 按钮点击事件，返回按钮类型 |

## 插槽

### 文本插槽
- `text`: 用于在按钮组前插入自定义文本内容

### 按钮插槽
- `button-{type}`: 用于自定义特定按钮的内容，其中 `{type}` 为按钮配置中的 type 值

```vue
<template>
  <CustomButton
    :btn-list="buttonList"
    @click="handleButtonClick"
  >
    <!-- 文本插槽 -->
    <template #text>
      <span>自定义文本:</span>
    </template>
    
    <!-- 自定义特定按钮内容 -->
    <template #button-add="{ btn }">
      <i class="el-icon-plus"></i>
      <span>{{ btn.text }}</span>
    </template>
  </CustomButton>
</template>
```

## 注意事项

1. 按钮权限控制依赖于 `v-has` 指令，请确保项目中已正确配置权限指令
2. 导出按钮基于 `TableDataExportButton` 组件实现，需要确保该组件可用
3. 每个按钮配置项必须包含 `buttonType`、`type` 和 `text` 字段
4. 导出按钮特有的属性仅在 `type` 为 `'export'` 时生效
